A futuristic, high-performance personal portfolio website featuring a **J.A.R.V.I.S-inspired 3D background**, **glassmorphism UI**, and seamless **Dark/Light theme** integration. Built with modern web technologies to showcase skills, projects, and services.

## âœ¨ Key Features

*   **J.A.R.V.I.S 3D Background:**
    *   Immersive, continuously rotating tech rings and hexagonal grids.
    *   Dynamic "Hubs" distributed across the screen (Top-Right, Center-Left, Bottom-Right).
    *   Floating scan lines, tech corners, and pulsing circles.
    *   Fully responsive and performance-optimized using CSS animations.
*   **Theme System:**
    *   **Dark Mode (Default):** Deep black/blue background (`#020617`) with neon cyan (`#22d3ee`) and pink (`#ec4899`) accents.
    *   **Light Mode:** Clean, high-contrast look with darker accent colors for visibility.
    *   **Persistence:** Automatically saves your theme preference to `localStorage`.
*   **Glassmorphism UI:**
    *   Modern, translucent cards with blur effects for "Tech Stack", "Services", and "Projects".
    *   Allows the animated background to shine through while maintaining text readability.
*   **Interactive Elements:**
    *   **3D Profile Photo:** Hover-activated 3D rotation and neon glow.
    *   **Neon Buttons:** Glowing hover effects and pulse animations.
    *   **Smooth Scrolling:** Seamless navigation between sections.

Follow these steps to run the project locally:

### Prerequisites

*   Node.js (v16 or higher)
*   npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/sarvesh9137/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
